Telecom Customer Churn Analysis

📌 Project Overview
This project analyzes customer churn in a telecom company using Python, SQL, and Power BI. The objective is to identify the key factors influencing customer churn, analyze customer behavior, and provide business insights through interactive dashboards.

🎯 Objectives
Analyze customer churn patterns.
Identify high-risk customers.
Understand factors affecting customer retention.
Build interactive Power BI dashboards for business decision-making.
